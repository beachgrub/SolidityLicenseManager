License Manager Example

Proof of concept for ERC721 LicenseManager Ethereum Contract running on the Rinkeby and Ropsten test networks.
You can claim unclaimed items. Assign them a daily rental rate in Ether. Also can rent claimed property for limited time.

For instructions on setting up Metamask to run on the test network see: Setting up Metamask
